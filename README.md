# 智汇知识库 Vue 前端原型

面向企业知识库/企业网盘场景的 Web 与小程序双端交互原型。界面参考“智慧园区”的克制型数字化风格：深海军蓝框架、白色内容区、企业蓝与青绿色状态色，不使用大面积渐变或玻璃拟态。

## 启动

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## 工程结构

- `index.html`：Vue、Iconify 与双端原型入口
- `kb-data.js`：响应式状态、模块数据和交互方法
- `kb-view-1.js`：Web 工作台、空间与文件交互模板
- `kb-view-2.js`：管理模块、小程序端和通用弹窗模板
- `styles.css`：全局框架和主题基础样式
- `theme.css`：列表、卡片、弹窗、小程序与响应式样式

所有模块位于同一个 Vue 3 根应用中，通过左侧菜单和“Web / 小程序端”切换按钮进入，无后端依赖。

## Iconify 在线图标

```html
<script src="https://code.iconify.design/iconify-icon/2.3.0/iconify-icon.min.js"></script>
```

通用操作图标使用 `solar` 图标集，Word、Excel、PDF 等文件类型使用 `vscode-icons` 图标集，不堆叠本地手写 SVG。

## 已实现的原型范围

- 我的工作台：空间统计、常用入口和待办
- 企业空间、个人空间：搜索、新建文件夹、上传、预览、外链和软删除
- 协作空间：项目卡片、状态和新增入口
- 与我相关：共享资源、权限审批和文件收集聚合
- 安全外链：权限、有效期和访问数据展示
- 误删恢复：保留期说明和还原操作
- 部门管理：组织节点、资料库用量和默认权限
- 成员管理：部门/岗位、账号状态与交接入口
- 管理员设置：管理范围和授权类型
- 空间统计、全局搜索和消息中心
- 小程序端：首页、文件、协作、消息和我的

原型数据保存在 Vue 内存状态中，刷新页面后恢复初始演示数据。
